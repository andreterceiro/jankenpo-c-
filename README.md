# jankenpo-c-\#

A simple Jankenpo game in C\#. I developed in Linux using Mono. Please see their website to install Mono ([Mono website](https://www.mono-project.com/))

# To compile

Using Mono:
`
csc jankenpo.cs
`

# To run
Using Mono, after compile:
`
mono jankenpo.exe
`
